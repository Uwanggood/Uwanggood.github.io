# Path Doctor SEO Site

Static GitHub Pages site for search traffic.

Canonical URL used in the HTML:

```text
https://uwanggood.github.io/
```

If this is deployed from a different repository name, replace that URL in:

- `*.html`
- `robots.txt`
- `sitemap.xml`

Pages included:

- `index.html`
- `ping-normal-site-slow.html`
- `tracert-star-star-star.html`
- `vpn-site-slow.html`
- `dns-normal-site-not-opening.html`

Deploy with the `Deploy SEO Site to GitHub Pages` workflow, then enable GitHub
Pages with GitHub Actions as the source in repository settings.
